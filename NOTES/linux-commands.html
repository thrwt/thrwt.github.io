Essential Linux Commands Cheat Sheet

Dec 05, 2025
3 min read
Study Notes

File Operations

Essential commands for working with files and directories:

Listing Files

# List files in current directory
ls

# List with details
ls -la

# List sorted by modification time
ls -lt

File Manipulation

# Copy file
cp source.txt destination.txt

# Move/rename file
mv oldname.txt newname.txt

# Delete file
rm file.txt

# Delete directory recursively
rm -rf directory/

Text Processing

Commands for searching and manipulating text:

# Search for pattern in file
grep "pattern" file.txt

# Search recursively in directory
grep -r "pattern" /path/

# Count lines in file
wc -l file.txt

# Display first 10 lines
head file.txt

# Display last 10 lines
tail file.txt

# Follow file updates in real-time
tail -f logfile.log

Network Commands

Useful networking commands for security testing:

# Check network connectivity
ping 8.8.8.8

# Show network interfaces
ifconfig
# or
ip addr show

# Display routing table
route -n
# or
ip route

# Check open ports
netstat -tulpn

# DNS lookup
nslookup domain.com
dig domain.com

User & Permissions

# Change file permissions
chmod 755 file.sh

# Change file owner
chown user:group file.txt

# Switch user
su - username

# Run command as root
sudo command

# Add user to group
usermod -aG groupname username

Process Management

# List running processes
ps aux

# Display processes tree
pstree

# Kill process by PID
kill 1234

# Kill process by name
pkill process_name

# Monitor system resources
top
# or
htop

Quick Tips

  • Use man command to read manual pages
  • Press Tab for auto-completion
  • Use history to view previous commands
  • Ctrl + R to search command history
  • Ctrl + C to cancel running command
  • Ctrl + Z to suspend process
